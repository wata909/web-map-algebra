## このページについて

このページは、科研費基盤研究(B)[スマート風土産業：ワイン専用品種の栽培適地評価による適地適作の実現（20H03121）](https://kaken.nii.ac.jp/ja/grant/KAKENHI-PROJECT-20H03121/)の紹介ページです
